常用权限管理命令

文件权限

  • 通过 ll 命令展示的列表
    • 以 d 开始的是目录文件
    • 以 - 开始的是普通文件
    • 以 l 开始的是符号链接
  • 文件权限
    • r 表示可读
    • w 表示可写
    • x 表示可执行
    • - 表示无权限

假设通过 ll 命令展示的文件列表中的某个文件的权限为 -rwxrwxrwx，那么：

  • - 表示文件类型
  • 第一个 rwx 表示当前用户的操作权限
  • 第二个 rwx 表示当前组的操作权限
  • 第三个 rwx 表示其他用户的操作权限
